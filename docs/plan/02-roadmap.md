## Phase 1 — 기본 홈페이지 오픈 (MVP)

> 회의 합의: "처음에는 심플하게, 기능 적게."

### 1-1. 정적 페이지

- **메인 (Top page)**:
  - 상단: 일러스트 메인 비주얼 (광안대교 캐릭터, 와이어프레임 그대로)
  - 브랜드 카피: 「釜山暮らし。いもじょも」
  - **3-pillar 카테고리 카드** (暮らす / 学ぶ / 旅する) ← Shizuoka 패턴
  - **New?** 신규 글 3개 카드 (콘텐츠 들어온 후 활성화)
  - **소코소코 모임 안내 CTA** (정보 사이트로만 보이지 않게)
  - SNS 링크 (Instagram / X / LINE / note)
- **About 페이지**: 모임 철학("건전한 관계 지향"), 활동 사진, 운영진
- **참여 안내**: 요일별 반(초·중·고급) 안내, 대기 신청 폼
- **Contact**: 이메일 / 인스타 / 네이버 카페 / X 링크

### 1-2. 정보 블로그 (Info Blog)

- 카테고리: 와이어프레임 3-pillar 그대로 (暮らす / 学ぶ / 旅する)
- 작성자 프로필 한 줄 (예: "부산 거주 N년 / ◯◯대학원 재학") → 신뢰도 부여
- 작성 주기: 주 1회 1건 이상 (운영진 10명 로테이션)
- AI 초안 작성 도구는 보조로만 사용

### 1-3. 다국어 (한국어 / 일본어)

- IP 기반 자동 + **헤더 우상단 JP|KR 토글**
- 콘텐츠는 처음부터 양 언어 동시 작성 원칙
- next-intl v4, `localePrefix: 'as-needed'`, 일본어가 default

### 1-4. SNS 자동 동기화

- 블로그 발행 시 X·Instagram 자동 포스팅 (API 연동)
- 공유 버튼 (X, Instagram, LINE, note) 모든 글에 노출
- ※ 인스타 그래프 API 제약 사항 사전 확인 필수

### 1-5. SEO 기반

- Google 우선 (sitemap.xml, robots.txt, structured data)
- 핵심 키워드: `釜山 日本語`, `釜山 留学`, `釜山 ワーホリ`, `釜山 暮らし`, `한국 유학`
- 글 작성 가이드라인: 제목 길이, 본문 분량(최소 800자), 이미지 alt, 내부 링크
- Google Search Console / Analytics 초기부터 연동

### 1-6. 인프라

- **Vercel Hobby** (무료, Next.js 16 1st party 지원)
- **임베디드 Sanity Studio** (`/studio`, 별도 배포 불필요)
- `@vercel/og` 동적 OG 이미지 + `next/image` 자동 최적화 (무료 한도 내)
- HTTPS(SSL) 자동, PR preview URL 자동
- Phase 2 광고/멤버십 도입 시 **Vercel Pro ($20/월)** 로 클릭 한 번 업그레이드
- 자세한 배포 절차: [`docs/runbooks/vercel-deploy.md`](../runbooks/vercel-deploy.md)

**Phase 1 종료 조건**: 운영진 20명 내부 베타 테스트 → 글 작성·다국어·SNS 동기화·검색 노출 확인 → 정식 오픈.

## Phase 2 — 커뮤니티 기능

### 2-1. 회원 등급 체계

| 등급                                  | 권한                | 시각 표시    |
| ------------------------------------- | ------------------- | ------------ |
| 비회원                                | 열람만              | —            |
| 일반 회원 (가입 + 이메일 인증)        | Q&A 질문·댓글       | 회색 아바타  |
| 정식 회원 (대표/운영진 승인)          | 자유 게시판 글 작성 | 🍎 사과 마크 |
| 인증 회원 (운영진·검증된 정보 제공자) | 정보 게시판 글 작성 | 👑 왕관 마크 |
| 관리자                                | 전체 권한           | 별 마크      |

### 2-2. 게시판

- **정보 게시판**: 인증 회원만 작성 / 누구나 열람
- **Q&A**: 일반 회원 이상 질문·댓글, 인증 회원 답변에 신뢰 마크
- **자유 게시판**: 정식 회원 작성, 후기·잡담
- **활동 게시판**: 운영진 작성, 모임 소식·이벤트

### 2-3. Q&A 부가 영역 (와이어프레임 확정)

- 우측 사이드바 **Rule 박스** — Q&A 작성 규칙
- 우측 사이드바 **Question Tips 박스** — 질문 잘 쓰는 법

### 2-4. 보안·법무 (필수)

- 개인정보 암호화 저장 (비밀번호 bcrypt 등)
- 개인정보처리방침 / 이용약관 게시 + 가입 시 동의 절차
- AI 기반 1차 스팸·욕설 필터 + 신고 기능
- 관리자 모니터링 대시보드 (회원 강퇴, 글 삭제)

> Phase 2 시점에 Supabase (Seoul), Vercel Postgres, Neon 등 DB 후보 검토 — [ADR-0003](../adr/0003-hosting-vercel.md) 의 "Phase 2 / Phase 3 인프라" 절 참조.

## Phase 3 — 체험 패키지 / 부가 가치

- **「소코소코 1회 体験」 페이지** (Trip 카테고리 하위에 진입점 배치)
  - 스터디 1회 참가 + 뒤풀이(식사) + 친구 매칭
  - 일정 신청 폼 + 운영진 응답 워크플로우
- 매년 재방문 일본인 친구 매칭 기능
- 연계 확장: 프리다이빙 등 운영진 개인 활동 → "다양한 삶의 기록" 게시판
