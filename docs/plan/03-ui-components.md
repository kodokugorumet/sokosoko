# 03. UI Components — 컴포넌트 명세

와이어프레임 v1 + 레퍼런스 4종(studyinshizuoka / hakko-blend / lumiarch / felissimo) 종합.

## 3.1 글로벌 헤더

**참고: studyinshizuoka.jp + hakko-blend.com**

```
┌─────────────────────────────────────────────────────────────┐
│ [釜山暮らし。いもじょも 로고]              [JP|KR][LOGIN][☰] │
└─────────────────────────────────────────────────────────────┘
```

- **현재 (Phase 1 초기)**: 카테고리 네비게이션은 MENU 드로어에 들어감 (와이어프레임 v1 그대로)
- **Phase 1 후반 추가 예정**:
  - 카테고리에 **호버 드롭다운** (Hakko-blend 패턴)
    - 学ぶ ▾ → 語学学校 / 大学 / 大学院 / VISA·ARC / 書類
    - 暮らす ▾ → 引っ越し / ゴミ / 免許 / 銀行 / Meet up
    - 旅する ▾ → Route / Food / Subway / KTX·ITX
  - **카테고리 라벨에 SVG 아이콘** (Hakko-blend 패턴) — 와이어프레임 일러스트 톤과 일관

- **JP|KR 토글** 위치: LOG IN **좌측** (와이어프레임 v1 → v2 보강 항목)

## 3.5 카드 컴포넌트

**참고: lumiarch + felissimo 합성**

```
┌────────────────────┐
│                    │
│   [썸네일 16:9]    │  ← 1024×576
│                    │
│ 🏠 暮らし    NEW   │  ← 카테고리 배지(이모지) + 신규 라벨(7일 이내)
│ 引っ越して。       │  ← bold 헤드라인
│ #釜山 #ワンルーム  │  ← 태그 (회색, 작은, 클릭 가능)
│ 👑 ケコ    2026.4.7│  ← 작성자(등급 마크) + 날짜(우측 하단 회색)
└────────────────────┘
```

**카드 동작**

- 카드 전체 클릭 → 글 상세 페이지
- 카테고리 배지 클릭 → 해당 카테고리 목록
- 태그 클릭 → 해당 태그 목록
- 작성자 마크 클릭 → 작성자 글 모음
- 호버: 약한 그림자 + 살짝 들림 (lift)
- 행 내 카드 높이 균등 (felissimo 패턴)

## 3.6 Q&A 페이지

**참고: 와이어프레임 그대로**

- 질문 카드: Date / Time / 댓글 수 메타
- 우측 사이드바: **Rule 박스** + **Question Tips 박스**
- 일반 회원 이상 작성, 답변자 등급 마크 노출

## 3.7 디자인 토큰 (현재 적용 중)

| 토큰            | 값                               | 용도                     |
| --------------- | -------------------------------- | ------------------------ |
| `--background`  | `#ffffff`                        | 페이지 배경              |
| `--foreground`  | `#1a1a18`                        | 본문 텍스트              |
| `--ink`         | `#111111`                        | 제목·강조                |
| `--paper`       | `#fafaf7`                        | 카드 배경                |
| `--accent`      | `#e8728c`                        | 마스코트 핑크 후드 컬러  |
| `--accent-soft` | `#fde7ec`                        | 강조 hover/배경          |
| `--muted`       | `#f5f5f2`                        | 약한 배경                |
| `--border`      | `#1a1a18`                        | 손글씨 박스 1.5px 외곽선 |
| `--font-hand`   | Klee One JP + Gaegu KR + Noto JP | 브랜드/타이틀에만 사용   |

> 손글씨 폰트는 **브랜드 워드마크와 페이지 타이틀에만** 적용. 본문은 시스템 sans-serif (가독성).
